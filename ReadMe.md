## Installation & Usage üìù

1. Clone the repository.
	```bash
	git clone https://github.com/kubinka0505/YTMVTG
	cd YTMVTG
	```
2. Install required modules by inputting pip install -r requirements.txt
3. Modify the parameters in the `Config.json`. [Its documentation can be found here](https://github.com/kubinka0505/YTMVTG/wiki/Configuration-Documentation).
4. Open shell script file named `Run`. Supports positional arguments - type `python __init__.pyw -h` for more.
5. Share Your image from the `Images` directory.
